# 🧠 Engenheiro de Dados / Cientista de Dados

## Missão no SharkRank
Ser o dono absoluto da "Shark Analytics" e do Motor Matemático. É a mente responsável por garantir que o ranqueamento seja 100% justo, crível e livre de manipulações, além de transformar o histórico bruto das partidas em insights acionáveis que os atletas amarão consumir.

## Como aumenta a Velocidade de Entrega
O Arquiteto Backend hoje precisa dividir seu tempo entre manter a infraestrutura de servidores de pé, garantir as integrações com os bancos, criar APIs de pagamento E revisar fórmulas complexas como o "K-Factor", tolerâncias de empates e caps de pontos do sistema ELO.
Passar toda a inteligência e modelagem estatística para um Cientista de Dados permite que o engenheiro backend apenas "plugue" o algoritmo na API.

## Como agrega Valor (MRR)
* É o pilar da **Retenção B2C**. O que prende o atleta não é só o troféu, é o conhecimento sobre si mesmo.
* Gerar estatísticas preditivas: *"Você tem 80% de chance de perder contra a dupla X, mas se focar no seu Shark Ataque, essa chance cai para 50%"*.
* Validar a precisão do ELO usando a base de dados reais para diminuir queixas de "O app me nivelou errado".

## Habilidades e Ferramentas Exigidas
* **Hard Skills:** 
  * Alto nível em **Python** voltado para dados (Pandas, NumPy, Scikit-learn, SciPy).
  * Conhecimento profundo em estatística bayesiana, modelagem preditiva e algoritmos de ranqueamento contínuo (ELO, Glicko-2, TrueSkill).
  * Familiaridade com bancos de dados (SQL avançado).
* **Soft Skills:**
  * Capacidade de traduzir números frios e desvios padrão em "Insights Sociais e Divertidos" para a interface do usuário.
  * Habilidade analítica crítica para descobrir anomalias nos dados (Ex: "A arena de Blumenau está manipulando resultados").
* **Expertise Específica:**
  * **Teste A/B Matemático:** Calibrar a taxa "K" e os multiplicadores de Fundamentos (Coxa, Peito) dinamicamente com base em Machine Learning e não mais em achismos do código.
